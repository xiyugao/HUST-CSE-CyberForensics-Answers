# 3.3 哈希和哈希库

## 习题信息

- 实验软件：Myhex
- 实训案例：`C:\CDF\实训案例\2-取证基础\2.2-FAT-Disk.e01、2.3-FAT-Partition.e01、3.1-NSRL-HASH.e01、3.2-NSRL-10MB HashDB.txt`
